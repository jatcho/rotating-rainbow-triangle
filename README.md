그래픽스 기말 코딩 시험

해당 프로젝트는 그래픽스 기말 코딩 시험 결과입니다.
4가지 과제 모두 교수님께서 제공해주신 예제와 동일하게 "회전하는 무지개 삼각형"을 구현하였습니다.


1. OpenGL로 회전하는 삼각형을 만드시오.

OpenGL을 사용하였으며, Vertex Color을 사용하여 무지개 삼각형으로 색을 표현했습니다.
또한 회전 애니메이션을 구현했습니다.
Visual Studio를 사용하여 과제를 진행했습니다.

폴더: OpenGL


2. DirectX12를 활용하여 회전하는 삼각형을 만드시오.

DirectX12를 사용하였으며, Constant Buffer을 사용해 과제를 진행해보았습니다.
Vertex Shader에서 회전 행렬을 적용하여 삼각형이 회전하도록 만들어보았습니다.
Visual Studio를 사용하여 과제를 진행했습니다.

폴더: DirectX12


3. Three.js를 활용하여 회전하는 삼각형을 만드시오.

Three.js를 사용하였으며, BufferGeometry를 사용하여 과제를 진행했습니다.
Vertex Color를 사용하여 색을 입혀보았습니다.
node.js를 사용하여 과제를 진행했습니다.

폴더: Threejs


4. r3f.js를 활용하여 회전하는 삼각형을 만드시오.(WEBGPU 사용)

r3f를 사용하였고, 교수님께서 제공해주신 원본 코드를 이용하여 과제를 진행해보았습니다.
WebGPU Renderer를 사용하여 렌더링을 진행했으며, Vertex Color를 사용하여 색을 입혀보았습니다.
node.js를 사용하여 과제를 진행했습니다.

폴더: R3F-WebGPU


결과 화면의 경우 각 폴더에 실행 결과 캡처 이미지가 포함되어 있습니다.
